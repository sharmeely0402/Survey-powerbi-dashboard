## Project Overview
An interactive Power BI dashboard created to analyze survey responses from data professionals worldwide.
The dashboard highlights job roles, salaries, preferred programming languages, work-life balance, 
demographics, and country participation.

## Key Insights
- Data Scientists report the highest average salaries
- Python is the most preferred language among Data Analysts & Scientists
- Majority of respondents are from the United States followed by India
- 74% male and 26% female gender participation
- Average age of respondents: ~30 years
- Moderate satisfaction with work-life balance across roles

## Tools & Skills
- Power BI Desktop  
- Power Query for data cleaning and transformation  
- DAX for calculated measures  
- Data visualization & dashboard design

## Dashboard Features
- Salary comparison by role
- Favorite programming languages by job profile
- Country-wise participant distribution
- Gender representation donut chart
- Work-life satisfaction gauge
- Ethnicity vs salary analysis
